Title: Moving Image Archive dataset
Description: This dataset represents the descriptive metadata from the Moving Image Archive catalogue (http://movingimage.nls.uk/) which is Scotland's national collection of moving image. The file format is zip, and contains 3 files: 1 plain text file containing the information on this page, 1 XML file of 6,017 records in MARCXML format (https://www.loc.gov/standards/marcxml/) and 1 XML file in Dublin Core format (https://dublincore.org/).
